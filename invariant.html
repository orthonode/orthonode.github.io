// FOUR_GATE_VERIFIER : EVERY_RECEIPT_PASSES_ALL_FOUR

Gate 1 — Identity
Agent Registry Check

agent_id must be in authorized registry. Blocks Sybil attacks and cross-miner output copying. The agent_id is hotkey-bound — cannot be forged.

Blocks: Sybil · Output Copying
Gate 2 — Model
Approved Model Registry

model_hash must match the community-approved model list. Blocks model impersonation — running a cheap model while claiming an expensive one.

Blocks: Model Impersonation
Gate 3 — Counter
Monotonic Counter

counter must be strictly greater than the last confirmed counter. Mathematically airtight replay and rollback prevention. No exceptions.

Blocks: Replay · Rollback
Gate 4 — Digest
SHA-256 Digest Verification

SHA-256(agent_id ‖ model_hash ‖ execution_hash ‖ counter) == digest. The execution_hash includes tempo_id — so cross-tempo caching fails this gate.

Blocks: Caching · Tamper
